
### 数据库 格式
#### 数据表
1. user [ 用户记录 ]
    * id        ( 用户ID ）
    * username  ( 用户名 )
    * nickname  ( 用户昵称 )
    * password  ( 用户密码 ）
    * sector    ( 用户所属部门 )

2.  records [ 返修数码记录 ]
    * id              ( 数码ID ）
    * buy_date        ( 购买时间 )
    * customer_name   ( 客户名称 )
    * customer_phone  ( 客户电话 )
    * brand           ( 品牌型号 )
    * string_code     ( 数码串码 )
    * appearance      ( 外观描述 )
    * screen          ( 显示屏 )
    * parts           ( 配件 )
    * digital_type    ( 数码类型 )
    * digital_color   ( 数码颜色 )
    * fault           ( 故障原因 )
    * remarks         ( 备注 )
    * repair_type     ( 维修类型 )
    * start_data      ( 送修日期 )
    * state           ( 状态 )
    * s_w_d           ( 门店送修仓库日期 )
    * w_m_d           ( 仓库送修厂家日期 )
    * m_w_d           ( 厂家返回仓库日期 )
    * w_s_d           ( 仓库返回门店日期 )
    * from_s          ( 数码来自的门店名称 )
